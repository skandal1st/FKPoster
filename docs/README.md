# HookahPOS — Документация

Облачная POS-система для кальянных, баров и ресторанов. Мультитенантная SaaS-платформа с поддержкой сетей заведений, поддоменной изоляцией и ролевой моделью доступа.

## Техническая документация (для разработчиков)

- [Архитектура](dev/architecture.md) — стек, структура проекта, мультитенантность, поддомены, WebSocket
- [API Reference](dev/api.md) — все эндпоинты, форматы запросов/ответов, коды ошибок
- [База данных](dev/database.md) — таблицы, связи, миграции
- [Запуск и разработка](dev/setup.md) — установка, переменные окружения, dev-режим
- [Деплой](dev/deploy.md) — Docker Compose, PM2, Nginx, SSL
