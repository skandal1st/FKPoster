# Администрирование

Все функции из этого раздела доступны ролям **владелец** и **администратор** (если не указано иное).

## Управление товарами

### Список товаров

Раздел **«Товары»** показывает все активные товары, сгруппированные по категориям. Для каждого товара отображается:
- Название
- Цена и себестоимость
- Текущий остаток
- Категория (цветная метка)

### Создание товара

1. Нажмите **«Добавить товар»**
2. Заполните поля:
   - **Название** — обязательно
   - **Категория** — выберите из списка (обязательно)
   - **Цена** — цена продажи
   - **Себестоимость** — для расчёта прибыли и маржинальности
   - **Единица** — шт, г, мл, порция
   - **Учёт остатков** — включите, если хотите отслеживать количество
   - **Минимальный остаток** — при достижении товар попадёт в отчёт «Низкий остаток»
   - **Штрихкод** — для сканера (опционально)
3. Нажмите **«Сохранить»**

### Редактирование

Нажмите на товар в списке → измените нужные поля → сохраните.

### Удаление (деактивация)

Товар не удаляется физически, а деактивируется (soft delete). Он перестаёт отображаться в каталоге и POS, но сохраняется в истории заказов.

### Составные товары (техкарты)

Составной товар — это товар, состоящий из ингредиентов. Например, кальян = табак + угли + одноразовый мундштук.

1. Создайте товар и включите **«Составной товар»**
2. Перейдите в раздел **«Техкарта»** товара
3. Добавьте ингредиенты:
   - Выберите ингредиент или группу ингредиентов
   - Укажите расход на порцию (например, 25 г табака)
4. Укажите **выход** (сколько порций из рецепта)
5. **Себестоимость** пересчитается автоматически

При продаже составного товара списываются ингредиенты (не сам товар).

### Низкий остаток

Товары с остатком ≤ минимального порога отображаются в отчёте **«Низкий остаток»** (в разделе товаров).

## Категории

1. Перейдите в раздел **«Категории»**
2. Каждая категория имеет:
   - **Название**
   - **Цвет** — для визуального выделения в POS
   - **Порядок сортировки** — влияет на порядок в каталоге
   - **Цех** (опционально) — привязка к цеху для печати тикетов

При регистрации создаются 3 категории по умолчанию: Кальяны, Напитки, Еда.

## Цеха (Workshops)

Цеха группируют категории по зонам приготовления (кухня, бар, кальянная).

1. Перейдите в раздел **«Цеха»**
2. Создайте цех (например, «Кухня», «Бар», «Кальянная»)
3. Привяжите категории к цехам (в настройках категории)

Цеха используются для:
- Разделения кухонных тикетов при печати
- Отчётов по выручке с разбивкой по цехам

## Ингредиенты

Ингредиенты — это товары, которые не продаются напрямую, но используются в техкартах.

1. Перейдите в раздел **«Ингредиенты»**
2. Создайте ингредиент:
   - **Название** (например, «Табак Al Fakher Двойное яблоко»)
   - **Категория**
   - **Себестоимость** — цена закупки за единицу
   - **Единица** — обычно граммы (г) или миллилитры (мл)
   - **Учёт остатков** — рекомендуется включить
   - **Группа ингредиентов** (опционально) — для взаимозаменяемых
3. Используйте ингредиенты в техкартах товаров

## Группы ингредиентов

Группа объединяет взаимозаменяемые ингредиенты. Например, группа «Табак для кальяна» может включать табак разных вкусов.

1. Перейдите в раздел **«Группы ингредиентов»**
2. Создайте группу:
   - **Название** (например, «Табак для кальяна»)
   - **Единица** (г, мл)
3. Привяжите ингредиенты к группе (в настройках ингредиента)

При использовании группы в техкарте:
- Списание происходит **пропорционально** остаткам членов группы
- Средневзвешенная себестоимость рассчитывается автоматически
- Суммарный остаток группы отображается в списке

## Поставки

Приёмка товара с автоматическим обновлением остатков и пересчётом себестоимости.

1. Перейдите в раздел **«Поставки»**
2. Нажмите **«Новая поставка»**
3. Заполните:
   - **Поставщик** — название поставщика
   - **Комментарий** (опционально)
4. Добавьте позиции:
   - Выберите товар
   - Укажите количество
   - Укажите цену за единицу (себестоимость)
5. Нажмите **«Сохранить»**

При приёмке автоматически:
- Увеличивается остаток товара
- Пересчитывается средневзвешенная себестоимость: `(old_qty × old_cost + new_qty × new_cost) / (old_qty + new_qty)`

## Инвентаризация

Сверка фактических остатков с системными.

1. Перейдите в раздел **«Инвентаризация»**
2. Нажмите **«Начать инвентаризацию»**
   - Система создаст список всех товаров с учётом остатков
   - Для каждого товара покажет системный остаток
3. Введите **фактические количества** для каждого товара
4. Нажмите **«Применить»**
   - Системные остатки обновятся по фактическим
   - Инвентаризация закроется

Одновременно может быть только одна открытая инвентаризация.

## Управление сотрудниками

1. Перейдите в раздел **«Сотрудники»**
2. **Добавить** — создайте нового сотрудника (имя + PIN или email + пароль)
3. **Редактировать** — измените имя, роль, PIN или пароль
4. **Деактивировать** — сотрудник не сможет входить

Доступные роли при создании:
- **Кассир** — работа с заказами
- **Администратор** — управление товарами, сотрудниками, отчёты
- **Владелец** — полный доступ

Также можно пригласить сотрудника по email (раздел **Настройки → Пользователи**).

## Гости и программа лояльности

### Создание гостя

1. Перейдите в раздел **«Гости»**
2. Нажмите **«Добавить гостя»**
3. Заполните:
   - **Имя**
   - **Телефон** (опционально)
   - **Тип скидки**: процент или фиксированная сумма
   - **Размер скидки**: например, 10% или 500 ₽
   - **Бонусный баланс** (опционально)

### Применение скидки

При оплате заказа кассир может выбрать гостя — его скидка применится автоматически к итогу.

### Статистика гостя

Для каждого гостя доступна статистика:
- Количество заказов
- Общая сумма заказов
- Общая сумма предоставленных скидок

## Настройки заведения

### Брендинг (только владелец)

- **Название** — отображается на экране входа и в отчётах
- **Логотип** — URL логотипа
- **Акцентный цвет** — фирменный цвет интерфейса (кнопки, выделения)

### Чеки (только владелец)

- **Ширина чека** — 58 мм или 80 мм
- **Шапка чека** — текст вверху чека (название, адрес)
- **Подвал чека** — текст внизу чека («Спасибо за визит!»)
- **Автопечать** — печатать чек при оплате автоматически

### Подписка

Просмотр текущего плана, лимитов и оставшегося времени. Смена тарифного плана.

### Интеграции (только владелец)

Настройка ЕГАИС (для алкоголя) и Честного знака (для табака):
- Включение/выключение интеграций
- Настройка подключения к УТМ (ЕГАИС)
- API-ключи (Честный знак)
