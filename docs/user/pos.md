# Работа с кассой

## Вход по PIN-коду

1. На экране входа отображается список сотрудников вашего заведения
2. Нажмите на своё имя
3. Введите 4-значный PIN-код на экранной клавиатуре
4. После ввода последней цифры вход произойдёт автоматически

Если PIN введён неверно — появится сообщение об ошибке. Можно попробовать снова.

Также доступна кнопка входа по email/паролю (для владельцев и администраторов).

## Кассовый день

Перед началом работы с заказами необходимо **открыть кассовый день**.

### Открытие кассового дня

1. При входе система покажет, что кассовый день не открыт
2. Нажмите **«Открыть кассовый день»**
3. Введите **сумму наличных в кассе** на начало дня (если начинаете с нуля — оставьте 0)
4. Нажмите **«Открыть»**

### Закрытие кассового дня

1. Перейдите в раздел кассового дня
2. Нажмите **«Закрыть кассовый день»**
3. Введите **фактическую сумму наличных** в кассе
4. Система покажет расхождение с ожидаемой суммой (если есть)
5. Нажмите **«Закрыть»**

Нельзя закрыть кассовый день, если есть незакрытые заказы — сначала оплатите или отмените все открытые заказы.

## Карта зала

На экране POS отображается карта зала со столами:

- **Свободный стол** — стол доступен для нового заказа
- **Занятый стол** — на столе есть открытый заказ (нажмите, чтобы открыть)
- Переключайте залы через вкладки (если несколько залов)

## Создание заказа

### Способ 1: Через карту зала

1. Нажмите на **свободный стол**
2. Выберите товар из каталога (фильтрация по категориям)
3. При добавлении первого товара стол автоматически станет **занятым** и создастся заказ

### Способ 2: Быстрый заказ (без стола)

Если работаете без привязки к столам — можно создать заказ без выбора стола.

## Работа с заказом

### Добавление позиций

1. Откройте заказ (нажмите на занятый стол)
2. Слева — каталог товаров, справа — текущий заказ
3. Нажмите на товар — он добавится в заказ (1 шт)
4. Повторное нажатие увеличивает количество
5. Итог пересчитывается автоматически

### Изменение количества

- **Кнопки +/−** — увеличить/уменьшить количество позиции
- При уменьшении до 0 — позиция удаляется из заказа

### Удаление позиции

Нажмите кнопку удаления рядом с позицией.

## Оплата заказа

1. Нажмите **«Оплатить»** (или кнопку оплаты)
2. Выберите способ оплаты:
   - **Наличные** — вся сумма наличными
   - **Карта** — вся сумма картой
   - **Смешанная** — часть наличными, часть картой (введите суммы вручную)
3. При необходимости выберите **гостя** для применения скидки
4. Нажмите **«Подтвердить оплату»**

После оплаты:
- Заказ закрывается
- Стол освобождается
- Остатки товаров автоматически списываются
- Итоги кассового дня обновляются

### Смешанная оплата

При выборе «Смешанная» необходимо указать:
- Сумма **наличными**
- Сумма **картой**

Сумма наличных + карта должна точно совпасть с итогом заказа.

### Скидки (через гостей)

1. При оплате нажмите **«Выбрать гостя»**
2. Найдите гостя по имени или телефону
3. Скидка применится автоматически (процент или фиксированная сумма)
4. Итог пересчитается

## Отмена заказа

1. Откройте заказ
2. Нажмите **«Отменить заказ»**
3. Подтвердите отмену

Отменить можно только открытый заказ. Оплаченные заказы отменить нельзя.

## Изменение способа оплаты

Если способ оплаты был указан ошибочно — его можно изменить для уже закрытого заказа:

1. Найдите заказ в списке
2. Измените способ оплаты
3. Итоги кассового дня автоматически пересчитаются

## Печать чеков

Если настроена печать (раздел **Настройки → Чеки**):

- **Ширина чека**: 58 мм или 80 мм
- **Шапка чека**: название заведения, адрес
- **Подвал чека**: «Спасибо за визит!» или своё сообщение
- **Автопечать**: чек распечатывается автоматически при оплате

## Маркировка (если включена)

Если включена интеграция с системой маркировки:

- При оплате заказов с маркированными товарами (табак, алкоголь) необходимо отсканировать все коды маркировки
- Система не позволит закрыть заказ, пока не будут отсканированы все требуемые коды

## Реальное время

Все изменения транслируются в реальном времени через WebSocket:
- Если другой кассир создал заказ — стол сменит статус на всех терминалах
- Если кассовый день открыт/закрыт — все увидят обновление
