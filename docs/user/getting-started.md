# Начало работы

## Регистрация заведения

1. Откройте главную страницу HookahPOS
2. Нажмите **«Регистрация»**
3. Заполните форму:
   - **Название компании** — название вашего заведения (например, «Дымная лаунж»)
   - **Ваше имя** — имя владельца
   - **Email** — для входа на главном домене
   - **Пароль** — минимум 6 символов
   - **Адрес** (slug) — адрес вашего заведения (опционально, генерируется автоматически из названия)
4. Нажмите **«Зарегистрировать»**

После регистрации:
- Создаётся ваше заведение с уникальным адресом (например, `dymnaya-launzh.hookahpos.ru`)
- Вы получаете роль **владельца** (owner) с полным доступом
- Активируется бесплатный пробный период на 14 дней
- Автоматически создаются категории: Кальяны, Напитки, Еда

## Первый вход

### Вход на главном домене (для владельца)

1. Откройте `hookahpos.ru`
2. Введите **email** и **пароль**
3. После успешного входа вы будете перенаправлены на поддомен вашего заведения

### Вход на поддомене (для сотрудников)

На поддомене заведения (например, `dymnaya-launzh.hookahpos.ru`) доступен быстрый вход по PIN-коду:

1. На экране входа отображается список сотрудников
2. Выберите своё имя
3. Введите 4-значный PIN-код на цифровой клавиатуре
4. Вход произойдёт автоматически после ввода четвёртой цифры

Также доступен вход по email/паролю (для владельца и администраторов).

## Создание сотрудников

1. Перейдите в раздел **«Сотрудники»** (боковое меню)
2. Нажмите **«Добавить сотрудника»**
3. Заполните:
   - **Имя** — как будет отображаться на экране входа
   - **PIN-код** — 4 цифры (должен быть уникальным среди всех сотрудников)
   - **Роль**: кассир, администратор или владелец
4. Нажмите **«Сохранить»**

Сотрудник сразу может входить по PIN-коду на поддомене вашего заведения.

## Настройка залов и столов

### Создание зала

1. Перейдите в раздел **«Залы»**
2. Нажмите **«Добавить зал»**
3. Укажите **название** (например, «Основной зал», «VIP», «Терраса»)
4. Настройте **размер сетки** (ширина × высота) — определяет разметку карты зала

### Добавление столов

1. Откройте нужный зал
2. Нажмите **«Добавить стол»**
3. Укажите:
   - **Номер** — уникальный номер в рамках зала
   - **Количество мест** — от 1 до 24
   - **Форма**: квадрат, прямоугольник, круг, угловой
   - **Метка** (опционально) — «VIP», «У окна» и т.п.
4. Перетащите стол на нужное место на карте зала

## Добавление товаров

1. Перейдите в раздел **«Товары»**
2. Нажмите **«Добавить товар»**
3. Заполните:
   - **Название**
   - **Категория** — выберите из списка
   - **Цена** — цена продажи
   - **Себестоимость** (опционально) — для расчёта маржинальности
   - **Единица** — шт, г, мл, порция
   - **Учёт остатков** — включить если нужно отслеживать количество
4. Нажмите **«Сохранить»**

## Настройка заведения

Перейдите в **«Настройки»** для:

- **Название** — обновить название заведения
- **Логотип** — загрузить логотип (отображается на экране входа)
- **Акцентный цвет** — фирменный цвет интерфейса
- **Настройки чеков** — ширина, шапка, подвал, автопечать
