# Отчёты и статистика

Все отчёты доступны в разделе **«Отчёты»** бокового меню. Большинство отчётов позволяют выбрать период (по умолчанию — последние 30 дней).

## Дашборд

Главный экран со сводной информацией за сегодня:

- **Выручка** — сумма оплаченных заказов за сегодня
- **Прибыль** — выручка минус себестоимость
- **Количество заказов** — закрытых за сегодня
- **Изменение выручки** — сравнение с тем же днём неделю назад (в процентах)
- **Открытые заказы** — сколько заказов сейчас открыто
- **Стоимость остатков** — общая стоимость всех товаров на складе
- **Низкий остаток** — количество товаров с критически низким остатком

Также на дашборде:
- **График тренда** — выручка за последние 7 дней
- **Топ-5 товаров** — самые продаваемые товары за сегодня
- **Продажи по категориям** — круговая диаграмма за сегодня

## Продажи

Детальный отчёт по продажам за выбранный период.

### Фильтры

- **Период** — от/до
- **Группировка** — по дням или по месяцам

### Данные

**Таблица и график:**
- Период (день/месяц)
- Количество заказов
- Выручка
- Наличные / Карта (разбивка)
- Себестоимость
- Прибыль

**Итоги за период:**
- Общая выручка
- Общая себестоимость
- Общая прибыль
- Итого наличными / картой
- Всего заказов

## Продажи по товарам

### Топ товаров

Рейтинг товаров по выручке за период:
- Название товара
- Количество проданных единиц
- Выручка
- Себестоимость

### Продажи по категориям

Круговая диаграмма распределения выручки по категориям:
- Название категории (с цветом)
- Выручка
- Количество проданных единиц

## Анализ себестоимости

Детальный анализ маржинальности каждого товара и категории:

**По товарам:**
- Название
- Количество продаж
- Выручка
- Себестоимость
- Прибыль
- Маржинальность (%)

**По категориям:**
- Та же разбивка, но по категориям

**Итоги:**
- Общая выручка
- Общая себестоимость
- Общая прибыль
- Средняя маржинальность

## Посещаемость

Анализ нагрузки заведения.

### По часам

График и таблица:
- Час дня (0–23)
- Количество заказов
- Выручка
- Средний чек

Определяется **пиковый час** — время с максимальным количеством заказов.

### По дням недели

- День недели (Пн–Вс)
- Количество заказов
- Выручка
- Средний чек

Определяется **самый загруженный день** недели.

### Общие показатели

- Всего заказов за период
- Общая выручка
- Средний чек
- Среднее количество заказов в день

## Статистика сотрудников

Эффективность каждого сотрудника за период:

- Имя сотрудника
- Количество заказов
- Выручка
- Средний чек
- Оплата наличными / картой

Отсортировано по выручке (от большей к меньшей).

## Анализ скидок

### Общая статистика

- Всего заказов
- Заказов со скидкой
- Общая сумма скидок
- Процент скидок от общей выручки

### Разбивка по гостям

Для каждого гостя:
- Имя
- Тип и размер скидки
- Количество заказов со скидкой
- Общая сумма предоставленных скидок
- Сумма оплаченная

## Отчёт по смене

Детальный отчёт по конкретному кассовому дню:

- Время открытия и закрытия
- Начальная наличность
- Выручка
- Прибыль
- Количество заказов
- Средний чек

**Список заказов смены:**
- Номер заказа
- Стол
- Кассир
- Способ оплаты
- Сумма
- Время

**Топ-5 товаров** за смену

**Почасовая загрузка** — график заказов и выручки по часам

**Выручка по цехам** — разбивка по кухне, бару, кальянной (если настроены цеха)

## Складские остатки

Отчёт по текущим остаткам (только для администраторов):

- Товар
- Категория
- Текущий остаток
- Минимальный остаток
- Единица
- Себестоимость за единицу
- Стоимость остатка (количество × себестоимость)
- Признак низкого остатка

**Итого**: общая стоимость всех остатков

Фильтрация по категории.

## Отчёты по сети (для владельца сети)

Если ваше заведение входит в сеть, доступны агрегированные отчёты:

### Дашборд сети

- Общая выручка, себестоимость, прибыль за сегодня
- Количество заказов по всей сети
- Средний чек
- Тренд за 7 дней
- Разбивка по заведениям

### Продажи по сети

- Период, группировка по дням/месяцам
- Общие итоги + разбивка по каждому заведению

### Сравнение заведений

Таблица с KPI каждого заведения:
- Выручка, себестоимость, прибыль
- Маржинальность
- Количество заказов, средний чек

### Топ товаров по сети

- Товар, категория
- Количество, выручка, себестоимость, прибыль
- Разбивка выручки по категориям
